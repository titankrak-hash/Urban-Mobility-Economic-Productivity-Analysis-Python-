# Business Objective

The objective of this project is to analyze how traffic congestion relates to economic productivity across major cities worldwide.

The analysis helps identify cities where transportation infrastructure investments could improve productivity and quality of life.

---

# Datasets

Two public datasets were combined.

### TomTom Traffic Index

Contains traffic indicators such as:

- Traffic Index
- Traffic Delay
- Travel Time
- Traffic Jams
- Average Delay

### OECD City Economy

Contains economic indicators including:

- GDP per Capita
- Unemployment Rate
- Population
- PM2.5 Air Pollution
